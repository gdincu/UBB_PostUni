<section class="slide">
<h2>Ce este Twitter Bootstrap?</h2>
<ul class="to5bottom to2right">
<li class="to2top"><span class="dimText">Front-End framework</span> pentru interfețe utilizator (<small class="dimText">Uis</small>) și teme</li>
<li class="totop">HTML/CSS/Javascript</li>
<li class="to2bottom">Folosit pentru a construi aplicații și site-uri web <span class="dimText">responsive</span> și <span class="dimText">mobile first</span></li>
</ul>
</section>


<section class="slide">
<h2>Istoria Bootstrap</h2>
<ul>
<li>dezvoltat de <a href="https://twitter.com/mdo" title="@mdo">Mark Otto</a> și <a href="https://twitter.com/fat" title="@fat">Jacob Thornton</a> la <a href="https://twitter.com">Twitter</a></li>
<li class="to1bottom">numit inițial <strong>Twitter Blueprint</strong>, a fost dezvoltat ca un cadru pentru a încuraja coerența între instrumentele interne.</li>
<li class="to2bottom">în 2010, după câteva luni de dezvoltare, Twitter a ținut <a href="https://blog.twitter.com/engineering/en_us/a/2010/hack-week.html">primul Hack Week</a>:
<ul class="small">
<li>mulți dezvoltatori de la Twitter au început să contribuie la proiect în cadrul săptămânii hackathon-style »</li>
</ul>
</li>
</ul>
</section>


<section class="slide">
<h2>Istoria Bootstrap. <strong class="small">Hackathon</strong></h2>
<h3 class="to2bottom">"hack" + "marathon"</h3>
<p class="em to2bottom">- un eveniment de tip sprint în care programatorii de calculator și alți experți din domeniu (de exemplu, designeri grafici, designeri de interfață, manageri de proiect) colaborează intens pe proiecte de software.</p>
</section>

<section class="slide">
<h2 class="to1top">Istoria Bootstrap (cont.)</h2>
<ul class="to2top small">
<li>redenumit de la <strong>Twitter Blueprint</strong> la <strong class="large">Bootstrap</strong>, și lansat ca un proiect <span class="dimText">open source</span> pe data de 19 august 2011</li>
<li class="to1bottom">La data de 31 ianuarie 2012, <strong>Bootstrap 2</strong> a fost lansat:
<ul class="small">
<li>a adăugat suportul încorporat pentru Glyphicons</li>
<li>mai multe componente noi</li>
<li>modificări ale multor componente existente</li>
<li>adăugarea funcționalității <span class="dimText strong">responsive</span> ca <span class="dimText">stylesheet</span> opțional</li>
</ul>
</li>
<li class="to2bottom"><strong>Bootstrap 3</strong>, a fost lansat pe 19 august 2013. Componentele au fost refăcute pentru a permite <strong class="dimText">flat design</strong> și <span class="dimText strong">responsive</span> în mod implicit cu
o abordare <strong class="dimText">mobile first</strong>.</li>
</ul>
</section>

<section class="slide">
<h2 class="to1top">Bootstrap - versiunea curentă</h2>
<ul class="small to2top">
<li><strong>Bootstrap 4</strong> a fost finalizat la 18 ianuarie 2018 și include schimbări semnificative:
<ul class="to4right">
<li>înlocuirea <a href="http://lesscss.org/">Less</a> cu <a href="https://sass-lang.com/">Sass</a></li>
<li>renunțarea la suportul pentru IE8, IE9 și iOS 6</li>
<li>permite CSS Layout Flexible Box, cunoscut sub numele de Flexbox</li>
<li>adăugarea opțiunilor de personalizare pentru navigație</li>
<li>adăugarea de distanțe și dimensionare <span class="dimText">responsive</span></li>
<li>trecerea de la unitatea de pixeli în CSS la <span class="dimText" title="este mărimea fontului elementului rădăcină al documentului">root ems</span></li>
<li>creșterea dimensiunii fontului global de la 14px la 16px</li>
<li>renunțarea la componentele <span class="dimText strong">panel</span>, <span class="dimText strong">thumbnail</span>, <span class="dimText strong">pager</span>, <span class="dimText strong">well</span></li>
<li>renunțarea la fontul Glyphicons</li>
</ul>
</li>
</ul>
<span class="copyright bottom">Documentație: <a href="https://getbootstrap.com/docs/">https://getbootstrap.com/docs/</a></span>
</section>

<section class="slide">
<h2 class="to1top">Viitorul Bootstrap</h2>
<strong>Bootstrap 5</strong> este viitoarea versiune majoră (probabil undeva în primul trimestru al anului 2020). Modificările majore includ:
<ul class="to1bottom to4right">
<li>trecerea de la biblioteca jQuery la JavaScript nativ</li>
<li>renunțarea la suport pentru IE10</li>
<li>schimbarea infrastructurii de testare de la QUnit la Jasmine</li>
</ul>
</section>

<section class="slide">
<h2 class="to1top">De ce să folosiți Bootstrap?</h2>
<ul class="to5right to2bottom">
<li class="to3right to3top">Creșterea vitezei de dezvoltare</li>

<li class="to3right"><span class="dimText">Customizable</span> (personalizabil)</li>
<li class="to3right to1bottom dimText">Support</li>
</ul>

<h2><span class="small">Instalarea mediului de lucru pentru Bootstrap</span></h2>
<ol class="small">
<li>verificarea existenței nodejs cu comanda: <code class="small">node -v</code>
<ul class="small">
<li>dacă nu există se poate aduce de la <a href="https://nodejs.org">https://nodejs.org</a></li>
</ul>
</li>
<li>crearea unui director nou folosind comanda: <code class="small">mkdir bs4 && cd bs4</code></li>
<li>crearea fișierului <em>package.json</em> pentru a salva dependințele din proiect <code class="small">npm init -y</code></li>
<li>instalarea pachetelor pentru dezvoltare: <code class="small">npm i gulp browser-sync gulp-sass --save-dev</code></li>
<li>instalarea pachetelor pentru proiect: <code class="small">npm i bootstrap jquery popper.js --save</code></li>
<li>pornirea editorului folosit (<small>de exemplu,</small> Atom) ... urmărirea <a href="https://coursetro.com/posts/code/130/Learn-Bootstrap-4-Final-in-2018-with-our-Free-Crash-Course">instrucțiunilor</a> </li>
</ol>
