De ce jQuery?

O modalitate ușoară de a realiza o varietate de sarcini comune JavaScript rapid și consistent, în toate browserele majore și fără a fi necesar cod suplimentar (fallback).
JavaScript and jQuery: Interactive Front-End Web Development, by Jon Duckett, July 2014 (@page: 294)

Ce este jQuery?

Permite căutarea elementelor cu ajutorul selectori de stil CSS și manipularea elementelor cu ajutorul metode jQuery.
JavaScript and jQuery: Interactive Front-End Web Development, by Jon Duckett, July 2014 (@page: 294)

Selectori de stil CSS
JavaScript and jQuery: Interactive Front-End Web Development, by Jon Duckett, July 2014 (@page 296)
A function called jQuery() lets you find one or more elements in the page. It creates an object called jQuery which holds reference to those elements.
$() is often used as a shorthand to save typing jQuery().

The jQuery() function has one parameter: a CSS-style selector.

.ready()

- specifică o funcție de executat atunci când DOM este încărcat complet

obiect jQuery metoda ready JavaScript and jQuery: Interactive Front-End Web Development, by Jon Duckett, July 2014 (@page 312)
$(document) creates JQuery object representing the page. When the page is ready the function inside the pharantesis of .ready() method is run.

.load() or .on('load') fires after the page and all its resources (images, CSS and scripts) have loaded. Recomandation: when one needs for instance the dimension of an image.

.ready() fires as soon as the DOM has loaded.

When the code is placed at the end of the page (before the closing </body> tag) the HTML will have loaded into the DOM before the script runs.
